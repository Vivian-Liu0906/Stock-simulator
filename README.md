# 📈 Stock Simulator

A web-based virtual stock trading platform built for high school students to learn investing through real market data.

## 🌟 Features

- 🔍 **Real-time stock data** — Search any stock by ticker symbol (e.g. AAPL, TSLA)
- 📊 **30-day price chart** — Interactive price history visualization
- 💰 **Virtual trading** — Start with $10,000 in virtual cash to buy and sell stocks
- 📂 **Portfolio tracking** — Track your holdings and average purchase price

## 🛠 Tech Stack

- **Backend:** Python, Flask, SQLAlchemy
- **Frontend:** HTML, CSS, JavaScript, Chart.js
- **Data:** yFinance API (real-time market data)
- **Database:** SQLite

## 🚀 Getting Started

### Prerequisites
- Python 3.x
- pip

### Installation

1. Clone the repository
```bash
   git clone https://github.com/yourusername/stock-simulator.git
   cd stock-simulator
```

2. Install dependencies
```bash
   pip install flask flask-sqlalchemy yfinance
```

3. Run the app
```bash
   python3 app.py
```

4. Open your browser and go to `http://127.0.0.1:5000`
